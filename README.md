# Laravel Api with Sanctum

## Specification

- Laravel 9
- PHP 8.1

## Installation

-  `git clone https://github.com/skcals/Laravel_api_auth_with_sanctum.git`
- `cd Laravel_api_auth_with_sanctum/`
- `composer install`
- `cp .env.example .env`
-  Update '.env' and set your database credentials
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan db:seed`
- `php artisan serve`
